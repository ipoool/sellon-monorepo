# SellOn Social Creatives — Style Guide

This guide defines the visual language for all SellOn social media creatives.
Use this as the reference when writing prompts for **nanobanana** image generation or creating visual briefs.

---

## Brand DNA (apply to all formats)

| Property | Value |
|---|---|
| Style | Clean Modern SaaS |
| Mood | Professional, trustworthy, warm, accessible |
| Primary color | Emerald Teal `#10b981` |
| Supporting colors | Emerald Deep `#059669`, Emerald Light `#d1fae5` |
| Neutral | Slate 900 `#0f172a`, Slate 700 `#334155`, Slate 100 `#f1f5f9`, White `#ffffff` |
| Accent | Amber `#f59e0b` (promo/highlight), Rose `#f43f5e` (urgency) |
| Background | White or Slate 100 as base; emerald teal as accent block |
| Shadow | Soft box shadow (blur, no hard offset) |
| Typography | Plus Jakarta Sans — Bold/SemiBold for headings, Regular for body |
| Illustration | Flat minimal, no heavy outlines, friendly but professional |
| Icons | Outline style, thin stroke (Lucide-style), emerald or slate color |
| Prohibited | Hard drop shadow, halftone dots, speed lines, comic colors (orange/purple/neon yellow), comic/rounded fonts |

## Format 1 — Instagram Post (Square 1:1)

**Canvas**: 1080 × 1080 px

### Layout Options

| Layout | Description |
|---|---|
| **Headline-first** | Large bold headline dominates 60% of the frame. Small outline icon or minimal illustration in the corner. Brand mark top-left or bottom. |
| **Split panel** | Left: illustration or UI mockup. Right: headline + subtext on a solid brand color block. |
| **Stat spotlight** | One large number or bold claim in the center (e.g. "0% commission"). Minimal supporting context below. |
| **Social proof** | Seller testimonial quote, avatar, star rating. White or slate-100 background. |

### Visual Rules
- At least one bold text element readable at thumbnail size
- Always include an emerald teal element as the visual anchor
- "SellOn" wordmark always present — bottom center or bottom-left
- Clear hierarchy: headline → subheadline → CTA/detail → brand mark
- UI/dashboard product screenshots are acceptable as the main visual (functional social proof)

### Prompt Template

```
[BASE PROMPT SNIPPET]
square 1:1 Instagram post, 1080x1080px,
[LAYOUT: headline-first / split panel / stat spotlight / social proof],
headline text: "[YOUR HEADLINE]",
[DESCRIBE MAIN VISUAL — illustration, UI mockup, or graphic element],
emerald teal #10b981 as dominant accent color, white or slate-100 background,
soft rounded containers, subtle shadow on cards,
"SellOn" wordmark bottom center in emerald teal
```

## Format 2 — Instagram Reels Cover (9:16 Vertical)

**Canvas**: 1080 × 1920 px

### Layout Options

| Layout | Description |
|---|---|
| **Hook top** | Bold hook text in the top 30%. Visual/illustration in the middle. CTA or logo at the bottom. |
| **Problem-solution** | Top: seller pain point (short text, slate/grey background). Bottom: SellOn solution (emerald teal background). Thin horizontal divider. |
| **UI showcase** | Dashboard/store screen mockup in the center. Short headline above. CTA below. |

### Visual Rules
- Hook text must be readable without sound in the feed preview
- Safe zone: critical content must not be within the top or bottom 250px (covered by UI)
- Use emerald teal as a background section or accent block, not as the full background
- Seller or buyer illustrations: simplified, friendly, representative

### Prompt Template

```
[BASE PROMPT SNIPPET]
vertical 9:16 Instagram Reels cover, 1080x1920px,
[LAYOUT: hook top / problem-solution / UI showcase],
bold hook headline at top: "[HOOK TEXT]",
[DESCRIBE CENTER VISUAL — seller illustration, UI mockup, or scene],
safe zone clear at top and bottom 250px,
emerald teal accent blocks, white dominant background,
"SellOn" wordmark bottom center
```

## Format 3 — Instagram Story (9:16 Vertical)

**Canvas**: 1080 × 1920 px

### Layout Options

| Layout | Description |
|---|---|
| **Feature highlight** | One feature per story card. Icon at top, headline in the middle, 1-line description, arrow CTA |
| **Seller tips** | "Tips" format — simple numbered list, clean visual |
| **Poll / Question** | Relatable question in center, interactive design, ready for IG poll sticker |
| **Promo / Offer** | Promo badge or large CTA in the center. Simple visual. Short copy. |

### Visual Rules
- Stories are interactive — design for tap/swipe behavior
- Max 6 words per text element
- High contrast between text and background
- Elements that can become stickers: badges, outlined icons, pill labels

### Prompt Template

```
[BASE PROMPT SNIPPET]
vertical 9:16 Instagram Story, 1080x1920px,
[LAYOUT: feature highlight / tips / poll / promo],
[DESCRIBE MAIN VISUAL OR CONTENT],
high contrast, white background with emerald teal accent elements,
large readable headline: "[TEXT]",
minimal text blocks max 6 words each, tap-affordance arrow or CTA element,
"SellOn" small brand mark top or bottom
```

## Format 4 — TikTok Thumbnail / Cover (9:16 Vertical)

**Canvas**: 1080 × 1920 px (cover), 1080 × 1350 px (in-feed crop zone)

### Layout Options

| Layout | Description |
|---|---|
| **Relatable problem** | Visual of an overwhelmed seller + relatable pain point text |
| **Number hook** | Large number dominates (e.g. "50 orders/day") + supporting visual |
| **Before/After** | Before (chaotic chats) vs. after (organized dashboard). Clear labels. |
| **Text-first** | A strong 3–5 word statement fills the frame, minimal background texture |

### Visual Rules
- Hook must be readable in under 1 second
- Must not look like an ad — native, relatable energy
- Believable Indonesian seller illustrations — not corporate stock photos
- Use TikTok caption style (white box, black text at the bottom) for familiar feel

### Prompt Template

```
[BASE PROMPT SNIPPET]
vertical 9:16 TikTok cover thumbnail, 1080x1920px,
[LAYOUT: relatable problem / number hook / before-after / text-first],
[DESCRIBE MAIN VISUAL — character expression or scene],
bold oversized hook text: "[HOOK TEXT]",
clean modern background emerald teal or white, minimal styling,
TikTok-style caption bar bottom: white rounded rectangle, black text: "[CAPTION]",
"SellOn" small brand tag top-right corner
```

## Do's and Don'ts

### Do
- Use generous white space — layouts that "breathe" feel more premium and trustworthy
- Make Emerald Teal the single dominant accent color
- Use illustrations and context relevant to Indonesian UMKM sellers
- Clear visual hierarchy: one most important element per composition
- Include the "SellOn" wordmark on every creative
- Use UI/dashboard mockups to demonstrate real functionality

### Don't
- Use hard drop shadow (offset without blur) — too comic-style, not SellOn brand
- Use orange, neon purple, or bright yellow as the dominant color
- Use comic/rounded fonts (Fredoka, Nunito, etc.)
- Add excessive decorative elements: halftone dots, speed lines, starburst graphics
- Make it look like a generic ad — visuals should feel relatable and native
- Use more than 3 colors in a single composition
- Use heavy element outlines (neubrutalism style)
- Use corporate stock photos that don't represent Indonesian UMKM sellers
